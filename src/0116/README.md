## 2주차 모각소 활동 느낀점
modifier 내 여러 함수들을 활용해보며 화면을 갖고 놀아봤다. 직접 그림 그린 것에 대해 코드로 화면에 구현할 수 있게 됐다. 하지만 실제 그림과 비슷하게 구현하는 데에 불편함이 너무 많은 것 같았다. 그림을 그리면 코드로 짜주는 ai가 나올 것 같은데 ai쪽으로 공부하고 싶다는 생각이 들었다.
